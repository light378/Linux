# Лабораторна робота 2
## Завдання 1
Написати програму для визначення моменту, коли time_t
закінчиться.

## Мета роботи 
- Дослідити, які зміни відбуваються в залежності від 32- та
64-бітної архітектури.
- Дослідити сегменти виконуваного файлу.

## Результати

### 64-бітна архітектура
```
light@vm:~$ gcc l2z1.c -o l2z1
light@vm:~$ ./l2z1
Size of time_t: 8 bytes
time_t is signed
Max time_t value: 9223372036854775807
```
### 32-бітна архітектура
```
light@vm:~$ gcc -m32 l2z1.c -o l2z1
light@vm:~$ ./l2z1
Size of time_t: 4 bytes
time_t is signed
Max time_t value: 2147483647
Overflow will occur at: Tue Jan 19 03:14:07 2038
```

виконуваного файлу, щоб отримати його загальний розмір, і
запустіть size, щоб отримати розміри сегментів всередині нього.
