## Завдання 5
Ні, у класичній фон-нейманівській архітектурі обійтися без лічильника команд, використовуючи замість нього лише вершину стека, неможливо. Однак існують стекові машини, де роль лічильника команд виконують інші механізми, але повністю відмовитися від збереження адреси повернення не вдається.
### Приклад
```
#include <stdio.h>

void foo() {
    int local_foo = 10;  // зберігається у стеку
    printf("In foo, local_foo = %d\n", local_foo);
}

int main() {
    int local_main = 5;  // зберігається у стеку
    printf("In main, local_main = %d\n", local_main);
    foo();               // виклик функції, адреса повернення зберігається у стеку
    return 0;
}
```
