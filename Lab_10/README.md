# Звіт (варіант 5)

Звіт по програмі, яка демонструє, як дочірній процес завершується з кодом 7, а батьківський читає цей код через `wait()`.

Файли
- `exit_l10z5.c` — C-програма, що реалізує fork, exit(7) у дочірньому процесі та читання коду виходу батьком через `wait()`.

Компіляція
```bash
gcc l10z5.c -o l10z5
```

Запуск
```bash
./l10z5
```

Опис поведінки
- Процес форкає дочірній процес.
- Дочірній процес виводить повідомлення та викликає `exit(7)`.

```
Child: exiting with code 7
```
